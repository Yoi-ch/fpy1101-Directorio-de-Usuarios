# Directorio de Usuarios
Proyecto CRUD para administrar usuarios mediante un menú interactivo. El sistema permite registrar, visualizar, actualizar y eliminar usuarios almacenados en una lista de diccionarios.

## Funcionalidades (CRUD)

Este proyecto incluye las siguientes funcionalidades:

* **Create**: Permite agregar un nuevo usuario al sistema validando que el RUT no se encuentre repetido.
* **Read**: Muestra la lista completa de usuarios registrados.
* **Update**: Modifica los datos de un usuario existente mediante la búsqueda por RUT.
* **Delete**: Elimina un usuario del sistema utilizando su RUT.

## Requisitos Técnicos Aplicados

* Uso de listas y diccionarios para almacenar la información.
* Uso de funciones para cada operación CRUD.
* Uso de estructuras de control (`if`, `for`, `while`).
* Validación de datos ingresados por el usuario.
* Validación de RUT y campos vacíos.
* Manejo de excepciones mediante `try/except`.
* Uso de `return` para finalizar funciones cuando corresponde.
* Menú interactivo con repetición mediante `while True`.
